# Installation

You could install it globally with `yarn`

```
yarn global add @vladimir-vovk/hangman-game
```

or `npm`

```
npm install -g @vladimir-vovk/hangman-game
```

Or run it with `npx`

```
npx @vladimir-vovk/hangman-game
```
